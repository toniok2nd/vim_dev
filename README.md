# To install on ubuntu
```bash
curl https://raw.githubusercontent.com/toniok2nd/vim_dev/master/template |bash
```

# To rebuild template
```bash
bash buildScript.sh
```
The script use local files:
- bashrcFile
- tmuxFile
- vimrcFile
